# CSCI 1300 Final Project

## Theme
Stardew

## Goal
fix community center within 7 days by 5 donating bundles before the week runs out
## How to compile and run
g++ final.cpp game.cpp player.cpp char.cpp item.cpp will compile everything and it can be run with a.out

## How to play
you will select your choices in a menu, spend energy/ gold, sleep / eat to replenish energy and work to get necessary items

## Classes

game.h and game.cpp is the main class that will run the game. It handles winning conditions and uses the other classes to play the game. player.h and player.cpp handle the player stats, like energy money and location. char.h and char.cpp handle the npc's names, friendship and quest progress. item.h and item.cpp handle item names, energy and whether or not an item has been donated as a winning condition.
